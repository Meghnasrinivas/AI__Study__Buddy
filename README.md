# 🤖 AI Study Buddy

## 🚀 Project Overview

AI Study Buddy is a full-stack web application that simulates an intelligent chatbot system. It enables real-time communication between users and a backend server, demonstrating API-based architecture and dynamic response handling.

This project reflects hands-on experience in building, debugging, and integrating a complete client-server system from scratch.

---

## 💡 Key Features

* 💬 Interactive chatbot UI
* 🔄 Real-time frontend ↔ backend communication
* 🌐 REST API integration using `fetch()`
* ⚡ Dynamic response rendering
* 🧠 Scalable architecture for AI integration
* 🔧 Error handling and debugging mechanisms
* 🔁 Asynchronous request handling

---

## 🔗 API & Concepts Used

* Fetch API (for sending HTTP requests)
* JSON data handling
* POST requests
* Client-server architecture
* Middleware usage (CORS, JSON parsing)

---

## ⚙️ How It Works (Flow)

1. User enters message in chat interface
2. JavaScript captures input and triggers `sendMessage()`
3. Frontend sends POST request to backend (`/chat`)
4. Backend receives request using Express server
5. Server processes request and prepares response
6. Response is sent back as JSON
7. Frontend dynamically updates UI with reply
